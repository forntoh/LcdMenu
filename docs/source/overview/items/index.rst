Item overview
=============

A menu item is a basic building block of a menu screen in |project|. Each menu item can contain zero or more widgets and can be interacted with by the user.

|project| provides a number of built-in menu items that you can use to create your menu structure.

You can also create your own custom menu items and widgets by extending the base menu item class or any of the existing menu items. See the :doc:`../widgets/index` section for more information about available widgets and their usage.
